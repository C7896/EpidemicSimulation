EpidemicSimulation Directory Contents:
	Directories
	- src (source code)
	- out (executables)
	- docs (all Javadoc)
	- manuals (user's and testing manuals)
	- scripts (running and javadoc scripts)

How to run scripts from the command line:
1) Open a terminal in the Battleship directory.
2) Run the following command:
	Windows:
		scripts\Run.bat <PATH_TO_JAVAFX\lib>
	macOS/Linux:
		bash scripts/Run.sh <PATH_TO_JAVAFX\lib>
